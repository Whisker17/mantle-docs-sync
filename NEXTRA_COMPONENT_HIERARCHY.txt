## MDX Component Registration Flow

mdx-components.js exports useMDXComponents()
    ↓
    ├── Imports getDocsMDXComponents() from nextra-theme-docs
    │   (provides: Callout, Steps, Link, etc.)
    │
    ├── Imports custom components from ./components/HomeCard.jsx
    │   (provides: Hero, CardGrid, HomeCard, etc.)
    │
    └── Merges into single object:
        {
          ...nextra_defaults,
          ...custom_mantle_components,
        }

Content author writes MDX:
    ↓
<Hero title="..." subtitle="...">
  <CTAButton href="..." variant="primary">Get Started</CTAButton>
</Hero>

MDX compiler looks up component in useMDXComponents()
    ↓
Finds Hero component → renders <div className="mantle-hero">
Finds CTAButton → renders <Link className="mantle-btn mantle-btn-primary">


## Sidebar Link CSS Selectors

### Distinguishing Internal vs External Links

<!-- Internal link (from _meta.js page reference) -->
<a href="/developers/quick-access">Quick Access</a>

<!-- External link (from _meta.js href property) -->
<a href="https://faucet.sepolia.mantle.xyz">Faucet ↗</a>

<!-- How to distinguish in CSS -->
a[href^="/"]         /* Internal link */
a[href^="http"]      /* External link */


### Current CSS for Sidebar

.nextra-sidebar-container a {
  transition: color 0.15s ease;
}

This applies to ALL sidebar links.


### Proposed Enhancement

/* All sidebar links */
.nextra-sidebar-container a {
  transition: color 0.15s ease, font-weight 0.15s ease;
}

/* Internal links - default style */
.nextra-sidebar-container a[href^="/"] {
  color: var(--mantle-text-primary);
}

/* External links - distinct style */
.nextra-sidebar-container a[href^="http"] {
  color: var(--mantle-primary);
  font-weight: 550;
}

.nextra-sidebar-container a[href^="http"]:hover {
  color: var(--mantle-primary-dark);
  text-decoration: underline;
}

## File Dependency Graph

next.config.mjs
  ├── Nextra wrapper configuration
  └── Points content to / directory

app/layout.jsx (Root)
  ├── Imports nextra-theme-docs/style.css (base)
  ├── Imports ./globals.css (overrides)
  ├── Imports { MantleLogo } from ./components/HomeCard
  └── Renders <Layout> with navbar, footer, sidebar

app/globals.css
  ├── Design tokens (:root)
  ├── Component styles (.mantle-*)
  ├── Nextra overrides (.nextra-*)
  └── Responsive media queries

mdx-components.js
  ├── Imports getDocsMDXComponents from nextra-theme-docs
  ├── Imports custom components from ./components/HomeCard
  └── Exports useMDXComponents() → merged registry

components/HomeCard.jsx
  ├── MantleLogo (SVG)
  ├── Hero
  ├── CTAButton
  ├── CardGrid
  ├── HomeCard
  ├── NavigationCard
  ├── DifficultyBadge
  ├── Expandable
  └── NavGrid

content/_meta.js
  ├── Root navigation structure
  └── Links to content/**/_meta.js files

content/[section]/_meta.js
  ├── Section-specific navigation
  ├── Internal page references
  └── External links (href properties)
