This tutorial teaches you how to use the Mantle Viem to transfer ETH between Layer 1 (L1) and Layer 2 (L2).&#x20;

### Setup

1. Ensure your computer has:
   1. `node`
   2. `npm/yarn`
   3. `typescript`
2. Install `mantle-viem` library, for example:

```bash
npm i @mantleio/viem
```

### Configuration of Viem Clients

First, we need to configure multiple clients, including the Sepolia network and wallet client, as well as the Mantle Sepolia network and wallet client, to facilitate subsequent function calls.


```text


```typescript
import \{
  publicActionsL2,
  walletActionsL1,
  walletActionsL2,
  publicActionsL1,
\} from "@mantleio/viem";
import \{ mantleSepoliaTestnet \} from "@mantleio/viem/chains";
import \{ createPublicClient, createWalletClient, http \} from "viem";
import \{ sepolia \} from "viem/chains";
import \{ privateKeyToAccount \} from "viem/accounts";

// Enter your own private key
export const account = privateKeyToAccount('0x...');

// Create Sepolia client
export const publicClientL1 = createPublicClient(\{
  chain: sepolia,
  transport: http(),
\}).extend(publicActionsL1());

// Create Sepolia wallet client
export const walletClientL1 = createWalletClient(\{
  account,
  chain: sepolia,
  transport: http(),
\}).extend(walletActionsL1());

// Create Mantle Sepolia client
export const publicClientL2 = createPublicClient(\{
  chain: mantleSepoliaTestnet,
  transport: http(),
\}).extend(publicActionsL2());

// Create Mantle Sepolia wallet client
export const walletClientL2 = createWalletClient(\{
  account,
  chain: mantleSepoliaTestnet,
  transport: http(),
\}).extend(walletActionsL2());
```

### Deposit ETH (From L1 to L2)

The deposit operation moves ETH tokens from L1 to L2. Here’s how to do it:

#### **Steps**

1. **Execute Deposit**
   1. Call the `depositETH` function to transfer tokens to L2.
2. **Wait for Confirmation**
   1. Confirm the transaction on both L1 and L2.

#### Code Example

```typescript
import "dotenv/config";
import { getL2TransactionHashes } from "@mantleio/viem";
import { mantleSepoliaTestnet } from "@mantleio/viem/chains";
import { publicClientL1, publicClientL2, walletClientL1 } from "./config.ts";
import { parseEther } from "viem";

console.log("Deposit Starts!!!!!!!");

// Step 1: Execute the deposit transaction on the L1.
const depositHash = await walletClientL1.depositETH({
  request: {
    amount: parseEther("0.01"),
  },
  targetChain: mantleSepoliaTestnet,
});
console.log("The deposit hash is:", depositHash);

//Step 2.1: Wait for the L1 transaction to be processed.
const depositReceipt = await publicClientL1.waitForTransactionReceipt({
  hash: depositHash,
});
console.log("The deposit L1 Tx is processed.");
//console.log("depositReceipt Log: ", depositReceipt.logs);

// Step 2.2: Get the L2 transaction hash from the L1 transaction receipt.
const [l2Hash] = getL2TransactionHashes(depositReceipt);
console.log("The deposit L2 Hash is:", l2Hash);

// Step 2.3: Wait for the L2 transaction to be processed.
const depositL2Receipt = await publicClientL2.waitForTransactionReceipt({
  hash: l2Hash,
});

console.log("The deposit L2 Tx is processed.");

console.log("Deposit Finished");
```

### Withdraw ETH (From L2 to L1)

The withdraw operation transfers ETH tokens from L2 back to L1. It involves multiple steps due to the bridge’s design.

#### **Steps**

1. **Approve Token Transfer**
   1. Since ETH works as an ERC-20 token on the Mantle Sepolia network, you need to set an allowance before depositing, which allows the L1 bridge contract to spend your ETH tokens.
2. **Initiate Withdrawal**
   1. Start the withdrawal process on L2.
3. **Wait for Proof**
   1. Wait until the withdrawal can be proven on L1.
4. **Prove Withdrawal**
   1. Submit the withdrawal proof on L1.
5. **Wait for Finalization**
   1. Wait until the withdrawal can be finalized.
   2. We manually added a wait time here (1 minute) to prevent withdraw failures due to prove information not being updated by the rpc.
6. **Finalize Withdrawal**
   1. Complete the process on L1.

#### Code Example

```typescript
import "dotenv/config";
import {
  account,
  publicClientL1,
  publicClientL2,
  walletClientL1,
  walletClientL2,
} from "./config.ts";
import { mantleSepoliaTestnet } from "@mantleio/viem/chains";
import { erc20Abi, maxUint256, parseEther } from "viem";

console.log("Withdrawal Starts!!!!!");

// This is the contract address for ETH token on Mantle Sepolia
const l2ETH = "0xdeaddeaddeaddeaddeaddeaddeaddeaddead1111";

// Step 1: Approve token transfer
const allowanceHash = await walletClientL2.writeContract({
  address: l2ETH,
  abi: erc20Abi,
  functionName: "approve",
  args: [mantleSepoliaTestnet.contracts.l2StandardBridge.address, maxUint256],
});
console.log("The allowance L1 hash is:", allowanceHash);
const allowanceReceipt = await publicClientL2.waitForTransactionReceipt({
  hash: allowanceHash,
});
console.log("The allowance L1 is processed.");

// Step 2: Execute the initiate ETH withdrawal transaction on the L2.
const withdrawalHash = await walletClientL2.initiateETHWithdrawal({
  account,
  request: {
    amount: parseEther("0.1"),
  },
});

console.log("The withdrawal L2 Tx hash is: ", withdrawalHash);

// Step 3.1: Wait for the initiate withdrawal transaction receipt.
const withdrawalReceipt = await publicClientL2.waitForTransactionReceipt({
  hash: withdrawalHash,
});

console.log("The withdrawal L2 Tx is processed.");

// Step 3.2: Wait until the withdrawal is ready to prove.
const { output, withdrawal } = await publicClientL1.waitToProve({
  receipt: withdrawalReceipt,
  targetChain: walletClientL2.chain,
});
console.log("The withdrawal L2 Tx is waiting for proving");

// Step 3.3: Build parameters to prove the withdrawal on the L2.
const proveArgs = await publicClientL2.buildProveWithdrawal({
  output,
  withdrawal,
});

console.log("The withdrawal prove Tx paras are built.");

// Step 4: Prove the withdrawal on the L1.
const proveHash = await walletClientL1.proveWithdrawal(proveArgs);

console.log("The withdrawal prove tx hash is: ", proveHash);

// Step 5.1: Wait until the prove withdrawal is processed.
const proveReceipt = await publicClientL1.waitForTransactionReceipt({
  hash: proveHash,
});
console.log("The withdrawal prove Tx is processed.");

// Step 5.2: Wait until the withdrawal is ready to finalize.
await publicClientL1.waitToFinalize({
  targetChain: walletClientL2.chain,
  withdrawalHash: withdrawal.withdrawalHash,
});
console.log("The withdrawal Tx is processed.");

// Step 5.3: Wait for rpc to update proving
const sleep = (ms: number) => new Promise((resolve) => setTimeout(resolve, ms));

console.log("Start waiting 1 minute...");

await sleep(1 * 60 * 1000); // 

console.log("Continue with the subsequent code...");

// Step 6.1: Finalize the withdrawal.
const finalizeHash = await walletClientL1.finalizeWithdrawal({
  targetChain: walletClientL2.chain,
  withdrawal,
});
console.log("The withdrawal Tx hash is: ", finalizeHash);

// Step 6.2: Wait until the withdrawal is finalized.
const finalizeReceipt = await publicClientL1.waitForTransactionReceipt({
  hash: finalizeHash,
});

console.log("The withdrawal Tx is finalized.");
```

### Summary

This guide provides the necessary steps and code to use Mantle Viem for depositing ETH from L1 to L2 and withdrawing it back to L1 via the official bridge. Always test your implementation in a test environment first. For additional details, consult [here](https://viem.mantle.xyz/).
