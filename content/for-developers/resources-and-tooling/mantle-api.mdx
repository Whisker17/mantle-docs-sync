---
title: "Mantle API"
---

Applications need to connect to a node to fetch block data and send transactions to the Mantle network. Mantle Network shares the same JSON-RPC API implementation as Ethereum.


> **Info**

You can refer to the common methods [here](https://ethereum.org/en/developers/docs/apis/json-rpc/#json-rpc-methods).

If you want to get the L2 gas price, you can either use [`eth_gasPrice`](https://ethereum.org/en/developers/docs/apis/json-rpc/#eth_gasprice) or get it from `GasPriceOracle` 's [`gasPrice`](https://mantlescan.xyz/address/0x420000000000000000000000000000000000000F#readProxyContract#F4) function. If you want to get the L1 gas price, you can get it from `GasPriceOracle` 's [`l1BaseFee`](https://mantlescan.xyz/address/0x420000000000000000000000000000000000000F#readProxyContract#F7) function.


We've also introduced custom methods to simplify certain Mantle-specific interactions. All methods can be accessed via `https` and `websocket`, endpoints can be found [here](/for-developers/resources-and-tooling/node-endpoints-and-providers).

## Custom Methods[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#custom-methods) <a href="#custom-methods" id="custom-methods"></a>

| **Method**          | **Description**                                                          |
| ------------------- | ------------------------------------------------------------------------ |
| `eth_getBlockRange` | Returns block info for multiple blocks, takes a block range as parameter |
| `rollup_getInfo`    | Returns L2 node info                                                     |

### `eth_getBlockRange`[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#eth_getblockrange) <a href="#eth_getblockrange" id="eth_getblockrange"></a>

Returns the block info in the form of an array of block objects for multiple blocks within a specified range. (See [`eth_getBlockByHash`](https://eth.wiki/json-rpc/API#eth_getblockbyhash) for the structure of a block object)

#### **Parameters**[**​**](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#parameters)

<table data-header-hidden><thead><tr><th width="209"></th><th width="173"></th><th></th></tr></thead><tbody><tr><td><strong>Parameter</strong></td><td><strong>Type</strong></td><td><strong>Description</strong></td></tr><tr><td><code>QUANTITY | TAG</code></td><td>Integer | String</td><td>Starting block no. of the range, or one of <code>"earliest"</code>, <code>"latest"</code>, or <code>"pending"</code>, as in the <a href="https://ethereum.org/en/developers/docs/apis/json-rpc/#default-block">default block parameter</a></td></tr><tr><td><code>QUANTITY | TAG</code></td><td>Integer | String</td><td>Ending block no. of the range, or one of <code>"earliest"</code>, <code>"latest"</code>, or <code>"pending"</code>, as in the <a href="https://ethereum.org/en/developers/docs/apis/json-rpc/#default-block">default block parameter</a></td></tr><tr><td><code>BOOLEAN</code></td><td>Boolean</td><td>If <code>true</code>, returns full transaction objects If <code>false</code>, returns transaction hashes only</td></tr></tbody></table>

#### **Sample Request**[**​**](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#sample-request)

```sh
curl -X POST --data '{"jsonrpc":"2.0","method":"eth_getBlockRange","params":["0x1", "0x2", false],"id":1}' <node url>
```

### `rollup_getInfo`[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#rollup_getinfo) <a href="#rollup_getinfo" id="rollup_getinfo"></a>

Returns various pieces of information in the form of an Object for a specified node. The response Object structure is as follows:

<table data-header-hidden><thead><tr><th width="205"></th><th width="130"></th><th></th></tr></thead><tbody><tr><td><strong>Field</strong></td><td><strong>Type</strong></td><td><strong>Description</strong></td></tr><tr><td><code>mode</code></td><td>String</td><td>Node's mode of operation - <code>"sequencer"</code> or <code>"verifier"</code></td></tr><tr><td><code>syncing</code></td><td>Boolean</td><td><code>true</code> if the node is currently syncing <code>false</code> otherwise</td></tr><tr><td><code>ethContext</code></td><td>Object</td><td>Nested object. Contains <code>blockNumber</code> and <code>timestamp</code></td></tr><tr><td><code>blockNumber</code></td><td>Quantity</td><td>Block no. of the latest known L1 block</td></tr><tr><td><code>timestamp</code></td><td>Quantity</td><td>Timestamp of the latest known L1 block</td></tr><tr><td><code>rollupContext</code></td><td>Object</td><td>Nested object. Contains <code>queueIndex</code>, <code>index</code>, and <code>verifiedIndex</code></td></tr><tr><td><code>queueIndex</code></td><td>Quantity</td><td>Index within the CTC of the last L1 to L2 message ingested</td></tr><tr><td><code>index</code></td><td>Quantity</td><td>Index of the last L2 transaction processed</td></tr><tr><td><code>verifiedIndex</code></td><td>Quantity</td><td>Index of the last transaction that was ingested from a batch that was posted to L1</td></tr></tbody></table>

#### **Parameters**[**​**](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#parameters-1)

None

#### **Sample Request**[**​**](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#sample-request-1)

```sh
curl -X POST --data '{"jsonrpc":"2.0","method":"rollup_getInfo","params":[],"id":1}' <node url>
```

#### **Sample Response**[**​**](https://docs-v2.mantle.xyz/devs/dev-guides/tools/api#sample-response-1)

  }
