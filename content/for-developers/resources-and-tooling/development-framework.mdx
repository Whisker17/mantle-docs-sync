## Mantle Own Librarys <a href="#hardhat" id="hardhat"></a>

### Mantle SDK

Mantle SDK is a fork of the `optimism-sdk` designed to support core functionalities of the Mantle network, such as deposits and withdrawals. For more details, please check this [page](/for-developers/how-to-guides/how-to-use-mantle-sdk).

### Mantle Viem

**Mantle Viem** is an extension package built on top of the [`viem`](https://viem.sh/) library, designed to simplify the development process on the Mantle network. For more details. please check this [page](/for-developers/how-to-guides/how-to-use-mantle-viem).

## Third-party Frameworks <a href="#hardhat" id="hardhat"></a>

### Hardhat[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/frameworks#hardhat) <a href="#hardhat" id="hardhat"></a>

[Hardhat](https://hardhat.org/hardhat-runner/docs/getting-started#overview) is a smart contract development environment that comes with tools that can be used to compile, debug, and deploy Solidity smart contracts and dApps on Ethereum and other EVM-compatible chains like Mantle Network.

### Foundry[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/frameworks#foundry) <a href="#foundry" id="foundry"></a>

[Foundry](https://book.getfoundry.sh/) is a fast, portable, and modular toolkit for EVM application development written in Rust. Foundry manages your dependencies, compiles your project, runs tests, deploys, and lets you interact with the chain from the command line and via Solidity scripts.

### Thirdweb[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/frameworks#thirdweb) <a href="#thirdweb" id="thirdweb"></a>

[Thirdweb](https://portal.thirdweb.com/) is a complete web3 development framework that provides everything you need to connect your apps and games to decentralized networks like Mantle Network, which will empower you to seamlessly build, launch, and manage web3 applications and games across any EVM-compatible blockchain.
