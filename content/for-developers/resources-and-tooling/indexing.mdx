In the blockchain field, to better understand and analyze blockchain data, often use Graph service or some Indexer service to process the data. Mantle Network supports a large number of data-related tools, providing users and developers with convenient data service infra.

## Graph Service[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#graph-service) <a href="#graph-service" id="graph-service"></a>

### Mantle Subgraph Studio[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#mantle-subgraph-studio) <a href="#mantle-subgraph-studio" id="mantle-subgraph-studio"></a>

[Subgraph](https://subgraph.mantle.xyz/) is a customized, open-source solution for indexing and accessing real-time blockchain data through the GraphQL API. They provide developers with powerful tools to build, customize and access blockchain data according to their needs. Check their [documentation](https://docs.ormi.xyz/dedicated-env/mantle/subgraphs/overview) for more details.

## Indexer[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#indexer) <a href="#indexer" id="indexer"></a>

### Lithosphere[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#lithosphere) <a href="#lithosphere" id="lithosphere"></a>

Lithosphere serves as the data hub of the Mantle Network, providing comprehensive data services to the infrastructure (bridge and explorer) and dApps, etc. in the ecosystem, by synchronizing the data from layer 1, layer 2, and Mantle DA, processing it before storing it in the main database, and ultimately exposing interfaces to the ecosystem through APIs.

### Birdeye Data Services

[**Birdeye Data Services (BDS)** ](https://bds.birdeye.so/)is a blockchain data infrastructure platform that provides fast and reliable multi-chain on-chain data through APIs and real-time streams. It aggregates token, trading, DeFi, and wallet data across major networks, enabling developers to easily integrate price feeds, analytics, and market insights into their applications.

### Covalent[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#covalent) <a href="#covalent" id="covalent"></a>

[Covalent](https://www.covalenthq.com/?utm_source=mantle\&utm_medium=partner-docs) collects and indexes historical data points from over 100 different chain networks and makes them available via one unified API. This can come in handy for developers working on multi-chain dApps. You can use Covalent if you need:

* Structured and enhanced on-chain data well beyond what you get from RPC providers
* Broad and deep multi-chain data at scale
* Enterprise-grade performance

### Dune Echo

[**Dune Echo**](https://dune.com/echo) is Dune’s new multichain real-time developer platform designed to provide real-time data processing and infrastructure support, enabling developers to efficiently build and scale blockchain applications.

* Echo consolidates data from **30+ blockchain networks** into a single API.
* Echo ensures that developers receive data within **300ms** of block propagation.

### Unmarshal[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#unmarshal) <a href="#unmarshal" id="unmarshal"></a>

[Unmarshal](https://console.unmarshal.io/) provides an API, TS, and Golang SDK, and a data parser to fetch indexed on-chain data and conveniently plug it into your dApps. You can also find analytics data available at a glance for multiple chains, including Mantle. Follow the link below to navigate to the Unmarshal console and get started.

### SimpleHash

[SimpleHash](https://simplehash.com/) is a platform that provides access to Token balances, DEX prices, and transfers, along with NFT metadata, media, collection details, sales, floor prices, listings and bids across multiple chains. SimpleHash now powers Mantle devs with real-time token data and transactions, full NFT metadata and balances and flexible access via API and bulk data streams.

### SubQuery[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/indexers#subquery) <a href="#subquery" id="subquery"></a>

[SubQuery](https://academy.subquery.network/quickstart/quickstart_chains/mantle.html) is a fast, flexible, and reliable open-source data indexer that provides custom APIs for dApps and web3 systems across multiple chains, including Mantle Network. Follow the link below to get started.
