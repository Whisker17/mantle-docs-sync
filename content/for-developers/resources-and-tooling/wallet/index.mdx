Mantle supports multiple wallets for interacting with the network.

## Reown[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/wallet/overview#web3modal) <a href="#web3modal" id="web3modal"></a>

Reown (formerly WalletConnect Inc.) is an onchain UX platform providing toolkits (AppKit and WalletKit) that help builders create intuitive, secure, and seamless Web3 experiences. Check more details [here](/for-developers/resources-and-tooling/wallet/reown).

## Multisig Wallet[​](https://docs-v2.mantle.xyz/devs/dev-guides/tools/wallet/overview#multisig-wallet) <a href="#multisig-wallet" id="multisig-wallet"></a>

Multisig wallet is a wallet that can be used to sign transactions with multiple signers. This is useful when you cooperate with multiple people in your project and also increase the decentralization of your project. Mantle also supports this important feature. Check more details [here](/for-developers/resources-and-tooling/wallet/multisig-wallet).
