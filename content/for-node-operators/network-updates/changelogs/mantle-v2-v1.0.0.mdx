* Add verification for some methods of bridge contract [#106](https://github.com/mantlenetworkio/mantle-v2/pull/106)

## op-geth[​](https://docs-v2.mantle.xyz/nodes/changelogs/v1.0.0#op-geth) <a href="#op-geth" id="op-geth"></a>
