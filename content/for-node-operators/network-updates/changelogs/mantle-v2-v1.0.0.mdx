Mantle v2 v1.0.0 is the confirmed version for the mainnet launch, which includes some bug fixes based on the testnet version [v0.5.0](https://docs-v2.mantle.xyz/nodes/changelogs/v0.5.0). The main updates are:

## Mantle v2 Tectonic 

* Enhance handling of abnormal situations in Mantle DA. [#117](https://github.com/mantlenetworkio/mantle-v2/pull/117)
* Add verification for some methods of bridge contract [#106](https://github.com/mantlenetworkio/mantle-v2/pull/106)

## op-geth 

* Optimize Meta Transactions UX. [#37](https://github.com/mantlenetworkio/op-geth/pull/37)
* Optimize the `estimateGas` function. [#67](https://github.com/mantlenetworkio/op-geth/pull/67)
