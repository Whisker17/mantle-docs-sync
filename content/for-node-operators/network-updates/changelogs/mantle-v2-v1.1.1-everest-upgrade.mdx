## Mantle v2[​](https://docs-v2.mantle.xyz/nodes/changelogs/v1.0.2#mantle-v2) <a href="#mantle-v2" id="mantle-v2"></a>

This release is primarily aimed at activating EigenDA on the Mantle mainnet, replacing the MantleDA. Additionally, this release includes compatibility preparations for the upcoming Ethereum mainnet Pectra upgrade.

* EigenDA Activation on the mainnet ([#204](https://github.com/mantlenetworkio/mantle-v2/pull/204))
* Compatibility preparations for the Ethereum Pectra upgrade ([#196](https://github.com/mantlenetworkio/mantle-v2/pull/196))

## op-geth[​](https://docs-v2.mantle.xyz/nodes/changelogs/v1.0.2#op-geth) <a href="#op-geth" id="op-geth"></a>

This release is primarily aimed at activating [all features of the Everest](https://github.com/mantlenetworkio/op-geth/releases/tag/v1.1.0) on the mainnet. Additionally, this release includes compatibility preparations for the upcoming Ethereum mainnet Pectra upgrade.

* Mainnet activation of Everest features ([#104](https://github.com/mantlenetworkio/op-geth/pull/104))
* Compatibility preparations for the Ethereum Pectra upgrade ([#101](https://github.com/mantlenetworkio/op-geth/pull/101))
