To build an app on Mantle network, you'll need access to a Mantle node. There are multiple public Node RPC Providers that you can choose from, or you can deploy your own depending on the requirements for your specific use cases.


<Callout type="info">
* **This doc is a simple introduction to the process.** For more details, please refer to this:
  * [Mantle v2 Tectonic Sepolia node](https://github.com/mantlenetworkio/networks/blob/main/run-node-sepolia)
* For those operators who have already deployed the Mantle Sepolia node, you can skip directly to this [section](#for-operators-who-have-already-deployed-nodes).
</Callout>


## For Operators Who Deploy Nodes For the First Time 

### Prerequisites 

Ensure you have the following software installed:

* [Git](https://www.git-scm.com/downloads)
* [docker](https://docs.docker.com/engine/install/)
* [node](https://nodejs.org/en/download/)
* [foundry](https://github.com/foundry-rs/foundry/releases)
* [zstd](https://github.com/facebook/zstd)
* [aira2](https://aria2.github.io/)

### Hardware Recommendations 

For optimal performance, it is recommended to have the following hardware specifications:

* **RAM** - 16GB+
* **CPU** - 8C+
* **Storage** - 500GB+ disk(HDD works for now, SSD is better)
* **Bandwidth** - 10mb/s+ download speed

### Operation Guide 

If you are a new operator who wants to deploy a Mantle Sepolia node, please follow the steps provided [**here**](https://github.com/mantlenetworkio/networks/blob/main/run-node-sepolia#installation-and-setup-instructions-for-new-user).
