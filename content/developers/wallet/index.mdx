Mantle supports multiple wallets for interacting with the network.

## Reown 

Reown (formerly WalletConnect Inc.) is an onchain UX platform providing toolkits (AppKit and WalletKit) that help builders create intuitive, secure, and seamless Web3 experiences. Check more details [here](/developers/wallet/reown).

## Multisig Wallet 

Multisig wallet is a wallet that can be used to sign transactions with multiple signers. This is useful when you cooperate with multiple people in your project and also increase the decentralization of your project. Mantle also supports this important feature. Check more details [here](/developers/wallet/multisig-wallet).
